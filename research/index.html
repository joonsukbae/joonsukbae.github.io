Research

My work spans ALICE Run 3 jet physics, central-framework software contributions to O2Physics, PWG-JE tracking-efficiency systematics for Hard Probes 2026, and Pb/scintillating-fiber calorimetry R&D for the ePIC Barrel Imaging Calorimeter at the Electron–Ion Collider.

Programme 01

ALICE Run 3 charged-particle jets

I am the lead analyzer of the first ALICE Run 3 charged-particle jet measurement (in preparation, target 2026): R-dependent charged-particle jet production in pp at √s = 13.6 TeV with ALICE. The analysis spans the full chain — detector-level QA on continuous-readout data, jet reconstruction in O2Physics, response-matrix building, Bayesian / SVD unfolding, and a complete systematic-uncertainty programme covering tracking effects and unfolding stability. Preliminary results are PWG-JE approved.

I established the Run 3-specific cross-section normalization based on the TVX visible cross section under continuous readout, and wrote jetCrossSectionEfficiency.cxx (the O2Physics task used by all PWG-JE Run 3 charged-jet analyses for normalization).

Programme 02

Tracking-efficiency systematics for Hard Probes 2026

I contributed the pp √s = 13.6 TeV total charged-particle tracking-efficiency systematic (~1.5% across the dominant pT range; pT-binned) for the PWG-JE Hard Probes 2026 preparation, including extension to the 2022, 2023, and 2024 periods. My ITS-TPC matching analysis code is adopted by a PWG-JE colleague for the corresponding PbPb tracking studies.

Related Run 3 understanding: trigger-level inefficiencies introduced by the continuous-readout TPC and asynchronous reconstruction; track-pT resolution studies (covariance-matrix propagation, MC residuals, V0 mass resolution) feeding a pT-dependent TrackTuner smearing now standard for charged-jet systematics in Run 3.

Programme 03

ePIC Barrel Imaging Calorimeter (BIC)

I am the lead analyzer for the on-site prompt and follow-up offline analysis of the CERN PS T10 (2025.07) Pb/scintillating-fiber beam-test paper (in preparation, 2026), and a co-author on the first beam-test paper (submitted to Nucl. Instrum. Meth. A, NIMA-D-26-00482; also submitted to JKPS; arXiv preprint posted).

Earlier hardware work: PMT–module QA prior to the test beams — signal characterization with cosmic-ray and radioactive-source data, timing-coincidence checks, optical-coupling stability, per-module acceptance criteria. On the simulation/reconstruction side I lead JANA2 / EIC-recon studies determining energy- and η-dependent sampling fractions and contribute Geant4 prototype modeling.

Software

ALICE O2Physics framework

I have contributed 35 pull requests (19 merged) to AliceO2Group/O2Physics since 2023, covering jet finding, response matrices, systematic-uncertainty pipelines, hadronic-rate fixes, and central reconstruction.

  • Central framework patch in Common/ (PR #15174, merged): fixed stale Q/pT values in graph-based TrackTuner momentum smearing — affects tracking systematics for all downstream analyses using TrackTuner.
  • Sole author of jetCrossSectionEfficiency.cxx (PR #14057, merged): the PWG-JE task used by all Run 3 charged-jet analyses for cross-section normalization at particle level.
  • Code review on 5 community pull requests across PWG-JE and Common.
Interests

What I am thinking about

  • Jet production and non-perturbative QCD: observables sensitive to hadronization and soft physics.
  • Jet substructure as a probe for quenching-like signatures in high-multiplicity pp and light-ion (OO, Ne–Ne) systems.
  • Initial-state effects and CGC-inspired frameworks for interpreting small-system jet measurements.
  • Detector R&D and performance characterization through test-beam data and simulation.
